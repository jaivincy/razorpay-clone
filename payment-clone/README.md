# PayFlow

An original, responsive fintech landing page built with HTML, CSS, and vanilla JavaScript.

## Run

Open `index.html` in any modern browser. No build process or dependencies are required.

## Included

- Sticky responsive navigation and mobile menu
- Animated dashboard, counters, scroll reveals, typing headline, and back-to-top control
- Testimonial carousel, FAQ accordion, pricing, payment suite, banking, and footer
