# 💳 PayFlow - Razorpay Inspired Payment Platform Clone

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/jaivincy/razorpay-clone.git
```

### Go to Project

```bash
cd payment-clone
```

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

Simply open

```
index.html
```

or use Live Server.
